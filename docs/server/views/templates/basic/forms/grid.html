Global variables

$form-input-font-family: inherit !global $form-input-border-radius: 0 !global $form-label-cursor: pointer !global $form-label-font-weight: 200 !global

Dont't forget to use for="#id-input-name" with <label>, it isn't necessary, but it's a good practice.

Creating a form is designed to be easy and flexible. Forms are built with a combination of standard form elements, as well as the Grid (-row and -col#). Form elements in Foundation are styled based on their type attribute rather than a class, and can be sized in a couple of ways:

  • You can size inputs using column sizes -col#
  • You can create row elements inside your form and use columns for the form, including inputs, labels and more. Rows inside a form inherit some special padding to even up input spacing.
To understand how to use Grid, please look at grid section.

Use theme classes (you can include any theme, so themes class names can be different) -primary-, -error-, -success-, etc. with <input> to set theme.

Use size classes Use size classes _tiny, _small, _minor, _normal, _major, _normal, _big, _huge, _divine with <input> to set size.
